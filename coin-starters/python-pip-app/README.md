# Python + pip app (Coin starter)

Минимальный Python-сервис. Golden path: `python-pip-app`.

```bash
cp -r coin-starters/python-pip-app/* /path/to/my-service/
# или: coin init --starter python-pip-app
```

CI: native venv + pytest → pack runtime-only Dockerfile. См. [docs/agent-build-model.md](../../docs/agent-build-model.md).
